
# Video Search
using (react - django RESTapi ) study project!

#세팅
docker를 실행할 수 있는 환경

# how to install
```bash
git clone https://github.com/tak-hirosi-reactproject-search/video_search.git
cd video_search
make run UID=<uid> USR=<user id> APP_NAME=video_search
```

# how to run
```bash
docker run <image name>
```
